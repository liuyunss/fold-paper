# 工艺折纸

## 项目来源

###

>爱好折纸，最初由折纸战士入坑，以为能折出里面的龙而自豪许久，直到我看到了龙神……
>由于一直在收集各种折纸教程，难免会有重复，所以想整理一下，也让自己看的时候更清晰点，
>但是整理下来发现，折纸之多，之最，格式之杂，所以先将收集的书籍整理了。
>目前放了比较著名的书
>如果你有想看的折纸书，可以留言告诉我，找到我会合到本项目里面。
>如果，还是说如果，如果日后有时间，我想重新捡起这个爱好。把想折的都折完。

###
